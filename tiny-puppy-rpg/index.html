Tiny Puppy RPG

This is a game I wrote in September of 2010.

It was written in ActionScript 3 (Flash), but these days, it's not possible to embed Flash in a web page anymore.

If you want to try this out, your best bet is to download an offline Flash player (requires ActionScript 3) and then download tiny-puppy-rpg.swf and run it.

Gave it a try? Let me know!
